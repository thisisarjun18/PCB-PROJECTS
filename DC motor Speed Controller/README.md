Overview

This project implements a PWM-based DC Motor Speed Controller using the NE555 Timer IC and a TIP122 Darlington Transistor.
The speed of the motor is controlled by adjusting the duty cycle of the PWM signal through a potentiometer, 
allowing efficient motor speed regulation without significant power loss.

Circuit Description

The NE555 timer is configured in astable mode to generate a PWM signal.
A potentiometer and two diodes are used to independently control the charging and discharging paths of the timing capacitor,
enabling duty-cycle adjustment while maintaining a nearly constant frequency.

The generated PWM signal drives a TIP122 transistor, which acts as a switching element for the DC motor. 
A flyback diode protects the transistor from back-EMF generated by the motor.

Working Principle
The NE555 generates a PWM waveform.
The potentiometer adjusts the duty cycle.
The PWM output drives the TIP122 transistor.
The transistor switches power to the motor.
Higher duty cycle → Higher average voltage → Higher motor speed.
Lower duty cycle → Lower average voltage → Lower motor speed.

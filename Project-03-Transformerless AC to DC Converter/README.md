Transformerless AC to DC Converter PCB Design

Overview

This project is a PCB design of a Transformerless AC to DC Converter created using KiCad. The circuit converts AC input into regulated DC output using a capacitive dropper, bridge rectifier, filter capacitors, and a voltage regulator.

The design demonstrates fundamental concepts of power electronics, PCB layout design, component placement, routing, ground plane implementation, and Design Rule Checking (DRC).

Features

* AC to DC conversion without a transformer
* Bridge rectifier using four diodes
* Capacitive dropper technique
* Voltage regulation
* Ground plane implementation on the PCB
* Designed using KiCad 10
* DRC verified with no electrical errors

## Components Used

* Terminal Blocks
* Capacitive Dropper Capacitor
* Resistors
* Bridge Rectifier Diodes
* Electrolytic Capacitors
* Ceramic Capacitor
* Voltage Regulator
* LED Indicator

PCB Design Highlights

* Single PCB design with optimized component placement
* Ground plane implemented using copper pour
* Through-hole components for easy assembly
* Electrical Rule Check (ERC) and Design Rule Check (DRC) completed

Learning Outcomes

This project helped in understanding:

* AC to DC conversion principles
* Bridge rectifier operation
* Capacitor filtering
* Voltage regulation
* PCB routing techniques
* Ground plane implementation
* KiCad schematic and PCB workflow

# Project 02 - Full Wave Bridge Rectifier

## Overview
A full-wave bridge rectifier PCB designed in KiCad to convert AC voltage into DC voltage using four 1N4007 diodes, a filter capacitor, and an LED power indicator.

## Components Used
- 4 × 1N4007 Diodes
- 1 × 1000µF Capacitor
- 1 × LED
- 1 × 2.2kΩ Resistor
- 2 × Screw Terminal Connectors

## Features
- Full-wave AC to DC conversion
- Capacitor filtering for smoother DC output
- LED power indication
- Through-hole components

## Tools Used
- KiCad 10

## Learning Outcomes
- Bridge rectifier operation
- AC to DC conversion
- Capacitor filtering
- Schematic design
- PCB layout and routing
